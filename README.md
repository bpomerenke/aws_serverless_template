# aws_serverless_template

This project gets the nuts and bolts of a terraformed serverless example up and running

# prerequisites
* Terraform
   * `brew install terraform`
* AWS CLI

# Editor / plugins
* Visual Studio Code
  * Terraform

# dev workflow
* `export AWS_PROFILE=<profile>`
* `./init.sh`
* `./deploy.sh`
* (OPTIONAL) `./teardown.sh`
